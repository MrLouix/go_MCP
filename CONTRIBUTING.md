# Contribuer au projet

Merci de votre interêt ! Voici comment contribuer.

## Workflow

1. Forkez le repo
2. Creez votre branche (`git checkout -b feature/MonNouvelleFonction`)
3. Commitez vos changements (`git commit -m 'feat: ajouter nouvelle fonctionnalite'`)
4. Pushez la branche (`git push origin feature/MonNouvelleFonction`)
5. Ouvrez une Pull Request

## Conventions de commit

- `feat:` nouvelle fonctionnalite
- `fix:` correction de bug
- `docs:` modifications de documentation
- `refactor:` refactoring sans changement fonctionnel
- `test:` ajout ou modification de tests
- `chore:` maintenance, dependances, CI/CD

## Pour commencer

```bash
npm install
npm run dev
```
